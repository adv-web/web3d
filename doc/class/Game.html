Class: Game

Defined in: src\Game.coffee

Overview

The game itself, takes a whole screen and render a scene.

The Game class is responsible to:

  • Initialize the game.

  • Take the game loop: in every frame update game logic and render it.

  • Handle DOM events.

Property Summary

(Scene) scene
the current scene of the game, you should use #setScene and avoid directly assign to it
