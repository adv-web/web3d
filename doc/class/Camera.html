Class: Camera

Overview

The camera component, a wrapper of THREE.Camera. The game should at least have a camera, or it will render a black screen.

name = "camera"

A camera on a game object will move with the game object's movement. To implement this, in Camera#afterAdded method set the camera as a sub-object of its game object's mesh. So as #beforeRemoved method.
