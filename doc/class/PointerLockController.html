Overview

The pointer lock controller component, provide users with first-person mouse movement. Modified by THREE.PointerLockControls to support when main object's mesh is not empty, so this can be used in both FPS and TPS.

name = "PointerLockController"

IMPORTANT: The component is related to a Camera component. Make sure you add this camera to game object before you add this controller component to game object, or the controller will not work properly.

When use pointer lock, you can request browser's better support. @see Game#requestPointerLock
