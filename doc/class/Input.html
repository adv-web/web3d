Class: Input

Defined in: src\Input.coffee

Overview

The static class that listens to key events.

You can register the onclick event. - registerClickResponse: (callback)

To read an axis use Input.GetAxis with one of the following default axes: "horizontal" and "vertical" are mapped to A, W, S, D keys.
