Web3d » Index » GameObject

Class: GameObject

Defined in: src\GameObject.coffee

Overview

The basic object in scene, can contain many components.

Property Summary

(boolean) isGameObject
to identify whether it is a game object, you should not change it
(THREE.Mesh | Physijs.Mesh) mesh
the mesh of the game object, especially, you can call mesh.gameObject to get the game object of mesh
(Object) components
the key-value pair of components and theirs name. e.g. {name1: component1, name2: component2}
