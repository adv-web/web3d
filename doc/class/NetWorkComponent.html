Class: NetWorkComponent

Defined in: src\component\NetWorkComponent.coffee
Inherits: Component

Overview

Base class which should be inherited by scripts which contain networking functionality.

It receive various callbacks help you to decide whether the player is a local player when in a multi player game. And it allows you to do something only to local players

Examples:

class MyComponent extends NetWorkComponent
  update: =>
    if @isLocal
      # do something
