Class: Scene

Defined in: src\Scene.coffee

Overview

The scene in game to contain game objects and other essential objects.

To render the scene, it will auto detect cameras and save it to an array when add a game object to the scene. But if there are multiply cameras in a scene, only the first one will be used to render.
